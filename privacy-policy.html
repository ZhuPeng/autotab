Privacy Policy for Smart Tab Manager

Last updated: 2025-01-24

Information Collection and Use

Smart Tab Manager collects and stores the following information locally on your device:

  • Tab access times for determining inactive tabs
  • Recently closed tab information (title and URL)

Data Storage

All data is stored locally using Chrome's storage API and is not transmitted to any external servers.

Data Retention

Data is only retained for the duration necessary to provide the extension's functionality and is automatically cleared when tabs are closed or the extension is uninstalled.
